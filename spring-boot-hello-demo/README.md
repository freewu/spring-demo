## 说明
```
SpringBoot Hello World Demo

使用 https://start.spring.io/ 生成
```

## 运行
```
# 安装依赖
    
    cd ./spring-boot-hello-demo
    mvn install
    
# 打包 

    mvn package
    
# 运行
    
    cd target
    java -jar .\spring-boot-hello-demo-1.0.0.jar
    
# 验证

    curl http://127.0.0.1:8888/hello
    
```
